Welcome

This is Super Smash Bros. StatX! Using the FrannHammer API, we bring you the character data for Super Smash Brothers WiiU/3DS. The character data is all real, and based off of data presented by KuroganeHammer, a moderator on Smashboards. All the data you see on this site originates from here along with character data being gathered from several other games! the FrannHammer API is currently under development, and so still receives updates from time to time. luckily, the updates for Smash 4 seem to be finished!

Under the Hood

The Characters page works by immediately calling on the API to populate the list of available characters (on the left in the full site, and in the dropdown on mobile), then when you select a character, the site runs a cross-reference on the json provided by the original call to find the URLs for the moves list and attributes list for the selected character. The page then makes two API calls for those lists and populates the tables you see on the page. The images for the characters are also provided by the API. The Versus page works in a similar fashion. the immediate API call populates the list of characters in both dropdowns, and when a character is selected, the image for them changes based on the json results of the initial call. When you click FIGHT!, the page randomly chooses who would win in a theoretical fight between these two characters.
